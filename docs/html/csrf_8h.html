Detailed Description

Модуль для обнаружения уязвимостей CSRF.

Предоставляет функциональность для сканирования веб-приложений на предмет Cross-Site Request Forgery (CSRF). Включает методы проверки защиты CSRF путём отправки запросов без cookies, Referer и Origin-заголовков.
