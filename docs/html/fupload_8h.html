Detailed Description

Модуль для тестирования уязвимостей при загрузке файлов

Пытается загрузить различные полезные нагрузки, чтобы обнаружить уязвимости загрузки файлов: обход фильтров типов, инъекции в имена файлов, загрузку исполняемых файлов и др.

void upload_file ( const std::string &  url)

Выполняет тестирование уязвимостей загрузки файлов на указанном URL.

Пробует различные полезные нагрузки (PDF, EXE, PHP-shell, двойные расширения, XSS в именах файлов) и сообщает, была ли загрузка принята или заблокирована.
